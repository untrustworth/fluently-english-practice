## 実際のテスト手順

1. **HTTPサーバー起動**:
   ```bash
   python3 -m http.server 8000
   ```

2. **ブラウザでアクセス**:
   `http://localhost:8000`

3. **Google AI Studio APIキー取得**:
   - https://makersuite.google.com/ でAPIキー生成
   - アプリの設定画面で入力

4. **各機能の順次テスト**:
   - 上記チェックリストに従って実行

## 既知の制約
- Web Speech APIはHTTPS必須（本番環境）
- Gemini APIは無料枠制限あり（15 RPM、1500 RPD）
- オフライン機能は限定的（PWAキャッシュのみ）
